# Sistema de Gestión de Gastos - Backend API

## Descripción

Aplicación web moderna construida con **Vue 3**, **Nuxt** y **TypeScript** para la gestión de gastos personales.

---

## Instalación

### Pasos

```bash
# 1. Clonar el repositorio
git clone <repository-url>
cd expenses-frontend

# 2. Instalar dependencias
npm install

# 3. Configurar variables de entorno
cp .env.example .env

# 4. Editar el archivo .env con tu configuración
nano .env

# 5. Iniciar el servidor de desarrollo
npm run dev
```

---

## Variables de Entorno

Usa el archivo `.env.example` como base y configura las variables necesarias:

```env
VITE_API_URL="ENLACE_DE_TU_API"
```
